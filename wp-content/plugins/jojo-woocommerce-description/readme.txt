== Description ==

This plugin allows you to show the short description of a WooCommerce product in product archives.
You can enable or disable the description under WooCommerce Settings Products. You can set a fixed height for the description adapt to the content.
You must have WooCommerce to use this plugin.
By setting the description height to a fixed value, you can align the products, so all the 'Add To Cart' buttons are aligned.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/jojo-woocommerce-description` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the WooCommerce->Settings->Products->JoJo WooCommerce Description screen to configure the plugin
